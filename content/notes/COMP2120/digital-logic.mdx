## Laws

| Law                 | Expression                                           |
| ------------------- | ---------------------------------------------------- |
| Identity Law        | $A + 0 = A$                                          |
|                     | $A \cdot 1 = A$                                      |
| Null Law            | $A + 1 = 1$                                          |
|                     | $A \cdot 0 = 0$                                      |
| Idempotent Law      | $A + A = A$                                          |
|                     | $A \cdot A = A$                                      |
| Complement Law      | $A + \overline{A} = 1$                               |
|                     | $A \cdot \overline{A} = 0$                           |
| Commutative Law     | $A + B = B + A$                                      |
|                     | $A \cdot B = B \cdot A$                              |
| Associative Law     | $A + (B + C) = (A + B) + C$                          |
|                     | $A \cdot (B \cdot C) = (A \cdot B) \cdot C$          |
| Distributive Law    | $A \cdot (B + C) = (A \cdot B) + (A \cdot C)$        |
|                     | $A + (B \cdot C) = (A + B) \cdot (A + C)$            |
| Absorption Law      | $A + (A \cdot B) = A$                                |
|                     | $A \cdot (A + B) = A$                                |
| De Morgan's Theorem | $\overline{A + B} = \overline{A} \cdot \overline{B}$ |
|                     | $\overline{A \cdot B} = \overline{A} + \overline{B}$ |

## Implementation of boolean function

We can implement any boolean function using either SOP or POS form.

We should use the **SOP** form if the function is mostly 1s, and **POS** form if the function is mostly 0s.

!!! tip Sum of Products (SOP)

    Uses the OR operation to sum multiple AND terms. Each AND term is called a **minterm** and represents a unique combination of variables that produce a true output (1) for the function.

    For example, the boolean function $F(A, B, C)$ can be expressed in SOP form as:
    $$ F(A, B, C) = \overline{A}BC + A\overline{B}C + AB\overline{C} $$

!!! tip Product of Sums (POS)

    Uses the AND operation to multiply multiple OR terms. Each OR term is called a **maxterm** and represents a unique combination of variables that produce a false output (0) for the function.

    For example, the boolean function $F(A, B, C)$ can be expressed in POS form as:
    $$ F(A, B, C) = (\overline{A} + \overline{B} + C)(A + \overline{B} + \overline{C})(A + B + C) $$

    Both SOP and POS forms are useful for simplifying and implementing boolean functions in digital logic circuits.
